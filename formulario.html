Formulario

Esta actividad permite la creacion de personas en la base de datos, tenga en cuenta que si desea agregar una nueva persona, los campos deberan contener valores, ademas el email debera ser uno valido como "ejemplo@xxx.com".

Para selecionar la sección a la que pertenece el monitor del que se almacenaran datos use el menu desplegable, si la deseada no se encuentra ahi puede hacer click en "AÑADIR" para que se habra una ventana de dialogo que le permitira registrar una seccion nueva que se guardare en la base de datos tan pronto como pulse "ACEPTAR" para guardar la persona actual. Tambien puede en todo momento pulsar en "CANCELAR" para borrar esta persona y volver a la lista
